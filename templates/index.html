Top Engineering Job Postings from Twitter - A RESTful Web Application by Joe Miao

Real-time Tweets search results served by Flask server will appear below, asynchronously. Meanwhile, they are parsed and converted to structed data, and finally saved into a relational DB to support job search features.

Stack: Twitter API(Rest&Streaming), Flask, SocketIO, SQLAlchmey, Jinja, Amazon EC2, Amazon RDS(MySQL)

Check /api to view available APIs. In addition, you can search jobs in db, e.g. api/postings?job_title=data
